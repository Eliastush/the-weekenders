# The Weekenders

A premium, mobile-first landing page for a Nairobi lifestyle community that helps people connect through weekend experiences.

**Meet people. Share experiences. Create memories.**

## What it is

The Weekenders is a Nairobi-based lifestyle community for social meetups, outdoor adventures, networking, game nights and road trips.

This website is a modern landing page designed to drive WhatsApp signups, highlight a featured community event, and present the brand as warm, premium, and easy to join.

## Landing Page Highlights

- **Hero section** with bold messaging, two CTAs and a lifestyle-inspired event preview
- **About section** that positions the brand as a lifestyle community, not a corporate events company
- **Experience section** showing core weekend offerings in clean cards
- **Featured event** spotlighting Uhuru Park Meet & Greet with premium CTA
- **Social proof** with community growth stats and trust messaging
- **Final strong call-to-action** to join WhatsApp

## Tech Stack

- **React 19**
- **TypeScript**
- **Vite**
- **CSS** with a brand palette and responsive mobile-first layout

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone <repository-url>
cd theweekenders
npm install
```

### Development

```bash
npm run dev
```

Open `http://localhost:5173` in your browser.

### Production

```bash
npm run build
npm run preview
```

## Structure

- `src/App.tsx` — root wrapper with WhatsApp quick action
- `src/pages/home.tsx` — premium landing page layout
- `src/App.css` — page styling, spacing, and component polish
- `src/index.css` — brand palette, typography, and reset styles
- `src/components/Icon.tsx` — lightweight SVG icon component

## Brand Style

- **Forest green** for the main brand tone
- **Warm beige** for background surfaces
- **Sunset orange** for the CTA accent
- **Soft rounded cards** and spacious sections
- **Mobile-first spacing** and large tap targets
